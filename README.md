Import numpy as np.

## Step 2:
Assign np.array() in inverse of a matrix.

## Step 3:
Using the np.linalg.inv() we can find the inverse of a matrix.

## Step 4:
Add the coding and end the program.


## Program:
```

import numpy as np
a=([[2,1,1],[1,1,1],[1,-1,2]])
```
